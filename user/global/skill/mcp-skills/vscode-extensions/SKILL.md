# VS Code Extensions for AI Development

## Core Extensions
1. **Trae AI** - AI coding assistant
2. **MCP Extension** - Model Context Protocol support
3. **GitHub Copilot** - AI pair programming

## Browser Automation
1. **Playwright** - Microsoft browser automation
2. **Puppeteer** - Headless Chrome control

## Development Tools
1. **Python** - Python language support
2. **C#** - .NET development
3. **Docker** - Container management

## Installation
\`\`\`bash
code --install-extension ms-playwright.playwright
code --install-extension ms-python.python
code --install-extension ms-dotnettools.csharp
\`\`\`
