# Playwright MCP Skill

## Description
Playwright MCP - Browser automation and testing toolkit

## Installation
\`\`\`bash
npm install -g @playwright/mcp
npx playwright install
\`\`\`

## Tools
- browser_navigate - Navigate to URL
- browser_screenshot - Take screenshot
- browser_snapshot - Get page snapshot
- browser_click - Click element
- browser_type - Type text
- browser_hover - Hover element
- browser_fill - Fill form
- browser_select - Select option
- browser_press - Press key
- browser_scroll - Scroll page
- browser_wait - Wait for element

## Usage
Use natural language to control browser:
- "Navigate to https://douyin.com"
- "Take a screenshot"
- "Click the login button"
- "Extract all video titles"
