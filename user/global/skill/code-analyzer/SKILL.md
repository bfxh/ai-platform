# 代码分析技能

## 概述

专业的代码分析和优化技能，支持多种编程语言。

## 功能

- 代码质量分析
- 性能优化建议
- 安全漏洞检测
- 风格检查
- 复杂度分析

## 用法

```python
from skills.code_analyzer import analyze_code, review_code

# 分析代码
result = analyze_code("path/to/code.py")

# 代码审查
review = review_code(code_string, language="python")
```

## 支持语言

- Python
- JavaScript/TypeScript
- Java
- C/C++
- Go
- Rust
