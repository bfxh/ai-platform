## [x] 任务 1: 强化 Blender MCP 核心功能
- **优先级**: P0
- **依赖**: 无
- **描述**:
  - 实现 HTTP 服务器，提供 REST API 接口
  - 实现自然语言命令解析，支持常见 Blender 操作
  - 添加完整的命令集：创建立方体、猴子头、设置材质、渲染场景等
  - 确保服务稳定运行
- **验收标准**: AC-1, AC-2
- **测试要求**:
  - `programmatic` TR-1.1: 发送 "创建立方体" 命令，验证 Blender 中是否创建了立方体
  - `programmatic` TR-1.2: 发送 POST 请求到 / 端点，验证返回结果格式正确
