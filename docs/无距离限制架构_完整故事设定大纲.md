# 完整故事设定大纲（无距离限制版）

---

## 一、起源：行星撞击与灵化

### 1.1 撞击事件
- 某行星/未知物体撞击世界
- 造成全球灾变，释放特殊能量
- 事件命名："大撞击" 或 "降临之日"

### 1.2 灵化本质
- 撞击能量感染部分人类/生物
- 灵化效果：
  - 身体变异/能量化
  - 觉醒特殊能力（如：召唤异界意识体）
- 主角在撞击中受伤，随后被灵化

## 二、主角人设：半机械体

### 2.1 身体状况
- 唯一原生部分：大脑（完整生物组织）
- 身体其余部分：机械义体
- 成因：撞击重伤 + 贫穷无力负担高级治疗
- 全部自己手搓：焊接废铁、拼装零件、改装遗弃设备

### 2.2 外貌参考
- 风格：灵笼·白月天
- 半张原生脸（放射性灼痕）+ 半机械
- 钛合金假眼（暗金冷光，可见光圈结构）
- 灰白乱发，全机械躯干无仿生蒙皮
- 胸腔透蓝光，手指为精密手术钳
- 气质：工业废墟感，机油/焊锡/臭氧混合气味

### 2.3 生存环境
- 贫穷偏远地区
- 无高科技，一切靠捡垃圾手搓
- 玩家接收器/信号器也需自搓

## 三、核心机制：穿越之躯

### 3.1 主角灵化能力本质
- 身体是维度通道
- 能将玩家意识从另一宇宙传输到本世界

### 3.2 传输原理
- 完整意识无法整体穿越维度壁垒
- 主角将意识拆解为无数灵魂切片，分批次、分频段传输
- 每片微小到足以"骗过"宇宙规则

### 3.3 传输铁三角
- 灵化者（主角）：活体通道
- 量子计算核心（服务器）：运算节点
- 生物质容器（培养舱躯体）：物质锚点
- 三者缺一，传输崩溃

### 3.4 数据传输流程
```
玩家意识 → 主角通道（维度传输，无距离限制）→ 服务器中转编译 → 主角回传 → 注入躯体
```

### 3.5 注入即自由（无距离限制核心）
- 灵魂切片注入躯体后，玩家成为独立灵化生命单元
- 不需要持续连接主角来维持存在
- 玩家可无限远离主角，世界完全开放
- 主角是"门"，不是"信号塔"——进门之后，去哪里是自己的事

### 3.6 差分传输
- 每片切片走不同频段路径
- 部分直传躯体，部分先入服务器
- 异步抵达后在服务器缓存中重组

### 3.7 烙印
- 每次传输完成，主角大脑承受负担
- 随机复制一小片玩家记忆/人格碎片
- 主角脑中逐渐沉淀"千万灵魂的碎片"
- 既是活体档案馆，也有被碎片逼疯的风险

### 3.8 命运编译
- 服务器对切片进行微观级调整
- 写入"命运轨迹代码"
- 投掷"二十一面命运骰子"决定天赋微量波动
- 因果锚定：让玩家与特定事件/人物产生非偶然联系

## 四、无距离限制下的五大替代机制

### 4.1 死亡回归绑定
- 玩家躯体被毁 → 意识自动切回VR端
- 重新召唤需主角重新传输
- 新躯体必须在培养舱制造，而培养舱在避难所
- 效果：远征可以，死了得从避难所重新跑图

### 4.2 信号覆盖率（替代距离）
- 距离不限制活动，但不同区域灵化信号强度不同
- 大撞击能量分布不均
- 高浓度区：连接稳定，躯体性能加成
- 零浓度区：躯体反应变慢，"手感变差"
- 死区：进入后躯体自动待机（可被拖出）

### 4.3 服务器中转延迟
- 玩家间通讯依赖服务器
- 距离越远，中转延迟越高
- 极端情况：远征队与基地通讯从秒级变分钟级
- 效果：用"信息延迟成本"替代"位置束缚"

### 4.4 烙印远程感应
- 每多一个玩家，主角脑中多一条模糊感应线
- 玩家情绪剧烈波动 → 主角感应到方向和情绪类型
- 玩家濒死 → 强烈刺痛，方向精确
- 主动共振可获取更多信息，但代价大

### 4.5 避难所唯一锚点
- 培养舱（复活点）在避难所
- 服务器在避难所
- 主角大脑（通道本身）在避难所
- 世界完全开放，但安全区只有一个

## 五、玩家职业三系

| 职业系 | 定位 | 特点 |
|--------|------|------|
| 机械系 | 战斗/改造专精 | 高攻高防，可加装武器模块，与主角半机械共鸣 |
| 生物系 | 回复/变异专精 | 自愈力强，可分泌活性物质，适应恶劣环境 |
| 普通系 | 基础平衡/成长型 | 无突出专长，适应性最广，成长上限灵活 |

### 5.1 玩家验证机制
- 第一层：信号覆盖范围限制（初期极小区域）
- 第二层：神经反馈特征筛选（拦截恶意/暴戾信号）
- 第三层：死亡强制断开保护

## 六、传输UI设定

### 6.1 核心理念
- 传输过程不是黑箱，是可见的精密手术
- 三端各有可视化呈现

### 6.2 三端视角

| 视角 | 能看到什么 | 呈现形式 |
|------|------------|----------|
| 主角端 | 灵魂切片流、大脑负载、烙印碎片、通道状态 | 神经直连·意识视野 |
| 服务器端 | 切片队列、命运编译进程、因果锚定坐标、纠错日志 | 实体屏幕/全息投影 |
| 玩家端 | 制造进度、切片重组动画、降临倒计时、躯体适配预览 | VR眼镜全息界面 |

### 6.3 主角端关键UI
- 通道状态环（蓝/琥珀/猩红 = 开启/传输/过载）
- 灵魂切片流（无数光粒涌入身体）
- 烙印碎片提示（传输完毕弹窗）
- 负载率表（超80%黄警，超95%红锁）

### 6.4 服务器端关键UI
- 切片队列面板（类似多轨编辑器，显示丢包率和预计重组时间）
- 命运编译进程（双螺旋光柱，金色代码注入）
- 因果锚定坐标图（三维星图式）
- 命运骰子动画（二十一面骰子投掷）
- 量子纠错日志（黄/橙/红三级错误）

### 6.5 玩家端关键UI
- 制造进度环（身体轮廓线稿逐步填充）
- 信号波形（底部滚动波形图）
- 职业预览卡（切换时轮廓变化）
- 心理承受值检测（显示评估结果和适应建议）
- UI个性化设置（允许玩家预设界面偏好）
- AI类型选择界面（辅助/战斗/探索型）
- AI特性预览（显示AI外观、能力和初始特性）
- 降临倒计时（归零白光覆盖全视野）

### 6.6 UI风格
- 核心视觉：工业废土 + 灵化能量
- 文字：等宽字体混手写体
- 图标：主角手绘 + 废弃设备截图
- 色彩：深灰黑底 + 琥珀橙/应急红/氚管绿/幽蓝荧光
- 质感：扫描线、信号干扰、像素破损

## 七、命运骰子与因果锚定（新增扩展）

### 7.1 命运骰子
- 每次传输编译完成时投掷
- 二十一面骰子
- 结果决定天赋参数微量波动
- 极稀有结果触发"天选"标记（隐藏剧情）

### 7.2 因果锚定
- 服务器在编译时写入
- 让玩家与特定事件/人物产生非偶然联系
- 三维星图可视化呈现
- 形成玩家独有的"命运线"

## 八、剧情主线流程

### 8.1 开篇：梦醒
- 反复梦到信息碎片、离奇死亡
- 从死亡梦中惊醒，发现已灵化
- 觉醒召唤玩家的能力

### 8.2 外出与追击
- 睡醒后外出，去往某地
- 被某小势力发现（灵化者身份暴露）
- 小势力一路追杀

### 8.3 诡异森林
- 被逼向诡异森林方向
- 追兵在森林边缘停止（非灵化者难以存活）
- 主角被迫深入

### 8.4 发现避难所
- 在山洞中仔细探查
- 发现隐藏的避难所（洞穴基地）
- 入口极隐蔽，或需灵化者身份

### 8.5 避难所设施

| 设施 | 功能 |
|------|------|
| 服务器 | 数据存储、切片编译、命运计算、召唤管理 |
| 培养舱 | 制造/修复玩家躯体 |
| 休眠舱 | 紧急休眠/生命维持 |
| 活性物质提取器 | 供能供材，维持主角大脑活性 |
| 核电系统 | 整体动力来源 |

### 8.6 造身体与调试（约1-2天）
- 根据职业调整身体参数
- 编写安全协议代码
- 修补、焊接、调试信号

### 8.7 测试玩家降临
- 主角放出少量信号测试
- 第一批玩家个位数
- 玩家群聊讨论真假

### 8.8 冲突展开
- 玩家降临后发展一段时间
- 与追杀势力开打
- 主角作为召唤者/指挥官参战

### 8.9 后续方向
- 修复避难所更多功能
- 侦查森林外围
- 反击追杀势力
- 扩大召唤规模
- 探索大撞击真相
- 可能建立第二基地（子服务器+培养舱）

## 九、可选模块：陪伴AI

### 9.1 若加入AI
- 载体：服务器
- 功能：管理设施、辅助召唤、解释来历、陪伴
- 交互：语音/全息/脑波连接

### 9.2 若不加入
- 主角自行探索日志/手册
- 氛围更孤独悬疑

## 十、核心公式总结

**注入即自由。主角是生门，不是锁链。**

- 传输需主角 + 服务器 + 培养舱（铁三角）
- 传输完成后玩家完全独立
- 约束靠死亡回归 / 信号覆盖 / 通讯延迟 / 烙印感应 / 避难所锚定
- 世界开放，根在山洞
