# ──────────────────────────────────────────────────────────

Godot 编辑器汉化:
  来源: Weblate 官方翻译平台
  文件: zh_CN.po → 放在 Godot 编辑器目录
  覆盖率: >95% 编辑器界面中文
  路径: Godot安装目录/editor/translations/

Godot 类参考汉化:
  来源: Weblate class-reference
  文件: classref_zh_CN.po
  用途: Godot内置文档中文

游戏内本地化:
  Godot内置 TranslationServer
  支持 .po / .csv / .gettext 格式
  GDScript代码: tr("KEY_NAME")

Blender 汉化:
  Edit → Preferences → Translation → 语言选简体中文
  Interface/Tooltips/NewData 三项全勾

ComfyUI 汉化:
  社区有 ComfyUI-zh_CN 翻译插件
  可在 ComfyUI Manager 安装
