# Release v3.1.0

Released: 2026-03-28

## Features

- **UMG Widget System** — 8 new MCP tools for building and managing UE UI widgets (create, modify, bind, screenshot)
- **Blueprint Graph Editing** — 14 new MCP tools for visual scripting node manipulation
- **Blueprint Discover Actions** — UE reflection-based action discovery
- **Blueprint Set Variable Default** — set CDO defaults via MCP
- **Blueprint Create Interface** — create Blueprint Interface assets
- **Blueprint Modify Component** — set component properties via reflection

## Improvements

- **Architecture Consistency** — race condition fixes, timeout coverage, N+1 query elimination
- **Logger Refactor** — functional Logger, dead code removal, improved Python error handling
- **DRY Refactor** — removed DRY violations, dead code, race conditions in listener
- **Dynamic Tool Registry** — runtime validation, type annotations, inputSchema validation
- **Stack Dependencies** — replaced flake8 with ruff, removed unused Python dependencies

## Security

- Hardened `python_proxy`, `console_command`, asset paths, CI supply chain
- Guard thread tracker, error handling, re-entrancy protections
- Reject empty path segments, command separators in asset paths
- Normalize env-var boolean checks to case-insensitive lowercase

## Bug Fixes

- Anchor float coercion and screenshot safety in widget tools
- Widget screenshot cleanup deferral with 17 unit tests
- Binding dedup coercion, docstring accuracy, anchors schema fixes
- Force-restart guard, callback safety, context validation
- Version sync across packages, path hardening
