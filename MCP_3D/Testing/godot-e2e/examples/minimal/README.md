# Minimal Example

The simplest possible godot-e2e setup. A single scene with one Label node and three tests demonstrating:

- **Node existence check** — verify nodes are in the scene tree
- **Property reading** — read a Label's text property
- **Method calling** — call a GDScript function and check the return value

## Run

```bash
godot-e2e tests/e2e/ -v
```
