# Contributing to AI Coding Assistant 🚀

First off, thank you for considering contributing to the AI Coding Assistant! It's people like you that make this tool better for everyone.

## 🛠️ How Can I Contribute?

### Reporting Bugs 🐛
- Use the **GitHub Issues** tab to report bugs.
- Include steps to reproduce, Godot version, and your OS.
- Attach any relevant error logs from the Godot output console.

### Suggesting Enhancements ✨
- Open an Issue with the tag `enhancement`.
- Describe the feature and why it would be useful.

### Pull Requests 🤝
1. Fork the repo and create your branch from `main`.
2. Keep your code clean and follow Godot's GDScript style guide.
3. Update the `CHANGELOG.md` with your changes.
4. Issue a PR and we'll review it together!

## 📜 Code of Conduct
Please be respectful and helpful to others in the community. Let's build something amazing together! 🎨🤖
