## Architecture
- Core Game: 2D Platformer with basic player movement, platforms, and collision.
- Player: CharacterBody2D with gravity, jump, and horizontal movement. Use logo.png as sprite.
- Platforms: StaticBody2D nodes for ground and obstacles.
- Main Scene: res://scns/main.tscn (new) as the game world.
- No enemies or UI yet – keep simple for demo.
- Physics: 2D, default settings.

## Naming Conventions
- Variables: snake_case
- Classes: PascalCase
- Files: snake_case

## Important Files
- res://scns/main.tscn: Main game scene.
- res://scripts/player.gd: Player script.
- res://scns/player.tscn: Player scene.
- res://scns/platform.tscn: Reusable platform scene.
- res://logo.png: Asset for player sprite.

## Current Goals
- Create a basic 2D platformer game using logo.png as the player's sprite.
- Set up player movement (walk, jump) with gravity.
- Add static platforms for the player to interact with.
- Make main scene playable.
- Test by running the project.
